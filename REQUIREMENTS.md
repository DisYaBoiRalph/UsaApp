# Requirements

## User Identification
- User login
    - app is now password protected
    - user can create their own profile with custom name and profile image
    - user can set their roles (student, teacher, other)

## Security
- Chat Room Privacy
    - User can set visibility or protection of chats
    - Either public or private where a client needs to provide a password
        that the host has set

## Notifications
- When a chat room is found, a notification fires that a notification is found
- When a user has joined the chat, a notification (either in notif bar or in the chat page) will also fire.

## Logging
- Latency logging can export to an actual CSV file
